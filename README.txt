Par soucis de ressources CPU et RAM, nous avons créé et utilisé un fichier XML allegé que l'on a appelé "dplb_light.xml" (présent dans ce dossier) contenant un échantillon (différentes publications des différents types) du fichier XML d'origine "dplb.xml".

Le programme de l'API tel qu'il est présenté, traite le fichier allégé et retourne un résultat relatif à ce fichier là.

Pour traiter le fichier XML d'origine, il suffit juste d'effectuer une petite modification dans le fichier source "API.py" en remplaçant la valeur de la variable file_name par le nom du fichier XML d'origine.
